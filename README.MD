# Portfolio

This workspace contains a frontend React portfolio scaffold in `fontend/`.

How to run the frontend locally:

```powershell
cd c:\Portfolio\fontend
npm install
npm run dev
```

Defaults used:
- React + Vite
- Styling: plain CSS in `src/styles/global.css`
- Animations: `framer-motion`
- Contact: Formspree form (no backend required)
- Suggested hosting: Vercel or Netlify
