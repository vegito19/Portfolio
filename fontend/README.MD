Portfolio frontend (Vite + React)

Quick start:

1. cd fontend
2. npm install
3. npm run dev

Notes:
- The contact form points to Formspree; replace the action URL with your Formspree form ID or wire to your backend.
- CV placeholder: drop a `cv.pdf` file into the `fontend` folder (served at `/cv.pdf`).
- Deploy to Vercel or Netlify for simple hosting.
